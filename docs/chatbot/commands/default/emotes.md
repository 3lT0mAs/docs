# !emotes

The `!emotes` command is used to display the list of available emotes in the chat.

### Usage

```
!emotes <platform>
```

#### Available Platforms

- `twitch`
- `bttv`
- `ffz`
- `7tv`

### Example Input

```
!emotes 7tv
```

### Example Output

```
Active 7TV emotes in chat: GIGACHAD JUSSY Joel JoelPride Joeler Joelest OMEGALUL donowall mamamia modCheck peepoLeave 
```
