# !chatstats

### Overview

The `!chatstats` command is used to get a link to the creators chatstats page.

### Module

- None

### Arguments

- None

### Example Input

```
!chatstats
```

### Example Output

```
https://stats.streamelements.com/c/styler PogChamp 
```
